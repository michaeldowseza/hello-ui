## Getting Started

## How To Install

### npm

```sh
npm install @michaeldowseza/hello-ui
```

### yarn

```sh
yarn add @michaeldowseza/hello-ui
```

## How To Use

```js
import { HelloWorld } from '@michaeldowseza/hello-ui'

const App = () => {
  return (
    <div>
      <HelloWorld />
    </div>
  )
}
```
